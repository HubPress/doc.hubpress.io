Developer documentation

Requirement

You need node, npm, bower and gulp. To build the documentation, Pygments is required.

Getting started

  1. Fork the repository https://github.com/HubPress/hubpress.io
  2. Clone your repository git clone
  3. Checkout the branch development
  4. Run npm install and bower install
  5. Run npm install -g gulp

To run the project :

  1. Edit the hubpress/config.json to the your username and the name of your repository
  2. Run gulp watch
